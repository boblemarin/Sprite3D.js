Sprite3D.js

a lightweight javascript library to simplify 3D positionning through CSS 3D transforms

Sprite3D.js makes it easy to manipulate HTML elements through CSS 3D transforms using a ActionScript-like syntax and hierarchy, a bunch of chainable accessor methods. A Sprite3D object is a simple wrapper around a DOM element providing helper functions and properties. You can conveniently use CSS styling to determine the aspect of you element, and CSS transitions to animate your stuff.

Dependencies

none

Install

Simply link to the Sprite3D.js library file and you are ready to go.
